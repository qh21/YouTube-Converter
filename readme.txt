- You can download YouTube videos as mp3 or mp4 via simple GUI.

- To use the YouTube converter download the YoutubeConverter.zip and unzip the folder
  then open (YoutubeConverter.exe) file.

- You can create a shortcut/s of the (YoutubeConverter.exe) file and put the shortcut anywhere on the PC.

- Copy The video link and paste it on the link area (the language of the keyboard must be in English).

- Some videos cannot be downloaded due to the YouTube terms.
